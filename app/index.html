😴 잠보의 방 — 탄수화물

"힘이 없어서 아무것도 못하겠어... 에너지가 하나도 없는데 뭘 먹어야 하지? 사탕이면 될 줄 알았는데 더 힘들어 😴"
📝 문제 1 / 3 — 객관식
탄수화물이 우리 몸에서 하는 주된 역할은 무엇인가요?
🍽️ 문제 2 / 3 — 단당류 / 다당류 분류 미션
잠보의 밥상에 있는 음식을 단당류와 다당류로 분류해 주세요!
🍇 포도당
🍎 과당
🍚 밥(녹말)
🥦 식이섬유
🍗 닭가슴살
🧈 버터
🔴 단당류
🟢 다당류
📜 문제 3 / 3 — 거짓말 찾기!
잠보의 방에서 비밀 일기장을 발견했습니다! 하지만 한 문장이 거짓말이에요. 찾아주세요!
📔 잠보의 비밀 일기
① 탄수화물은 우리 몸의 주된 에너지원이다.
② 밥, 빵, 감자에는 다당류(녹말)가 풍부하다.
③ 단당류는 천천히 소화되어 에너지가 오래 지속된다.
④ 탄수화물이 부족하면 기운이 없고 집중력이 떨어진다.
💡 단당류(포도당·과당)는 빠르게 흡수되어 혈당이 급등했다 급락합니다. 오래 지속되는 것은 다당류예요!

🧑‍🦳 저리의 방 — 무기질

📝 문제 1 / 3 — 객관식
뼈와 이를 튼튼하게 만드는 무기질은 무엇인가요?
🛒 문제 2 / 3 — 장바구니 미션

🍭 사탕
🍔 햄버거

🛒 장바구니 (3개를 담으세요)
🔎 문제 3 / 3 — 표 오류 찾기!
저리의 책상에서 무기질 정리표를 발견했습니다! 단 하나의 행이 틀렸어요. 잘못된 행을 클릭하세요!
무기질주요 역할결핍 시 증상
칼슘 (Ca)뼈와 이를 구성결핍 시 뼈 약화 (구루병, 골다공증)
철분 (Fe)헤모글로빈 구성 (산소 이동)빈혈, 피로
나트륨 (Na)⚠️ 칼슘 흡수 촉진뼈 성장 장애
아연(Zn)·마그네슘(Mg)성장과 발육을 돕는 여러 기능성장 지연, 경련 등
💡 나트륨(Na)의 역할은 세포의 삼투압 조절(수분 평형)이에요! 칼슘 흡수 촉진은 비타민 D, C, 젖당입니다.
