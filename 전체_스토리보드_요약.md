## 🚪 3. 각 방별 스토리 및 문제 구성

### 🍚 방 1: 잠보 (탄수화물) → 코드 획득: [영]
* **대사:** "너무 힘이 없어서 아무것도 못 해... 맨날 설탕 듬뿍 과자만 먹었는데 😵‍💫"
* **Q1. 객관식:** 탄수화물의 주된 역할은? (정답: 우리 몸을 움직이는 가장 중요한 에너지원)
* **Q2. 드래그 앤 드롭:** 단당류(설탕)와 다당류(녹말) 카드 2개씩 알맞은 바구니에 드래그하여 분류하기.
* **Q3. 거짓말 찾기 (방탈출):** 잠보의 비밀 일기장에서 틀린 문장 찾기.
  * (정답: ③ 단당류는 천천히 소화되어 에너지가 오래 지속된다 → 단당류는 빠르게 흡수되어 혈당을 급등락시킵니다.)

### 🥩 방 2: 야위 (단백질) → 코드 획득: [양]
* **대사:** "무거운 것도 못 들고... 근육이 왜 이리 없지? 고기도 잘 안 먹고 과자만 먹었는데... 😢"
* **Q1. 객관식:** 단백질이 우리 몸에서 주로 하는 역할은? (정답: 근육, 피부, 머리카락 등 몸의 구성 성분)
* **Q2. 일기 빈칸 채우기:** "의사 선생님이 `단백질`이 부족해서 `근육`이 줄었다고 했어. 앞으로는 `달걀·고기·콩`을 많이 먹어야지!"
* **Q3. 문장 조합하기 (방탈출):** 필수 아미노산의 비밀 문장 순서대로 맞추기.
  * (정답: `필수 아미노산은` → `체내에서 합성되지 않아` → `반드시` → `음식물로 섭취해야 해!`)

### 🫒 방 3: 비실 (지방) → 코드 획득: [전]
* **대사:** "항상 춥고 피부가 건조해... 지방은 나쁜 거라고 생각해서 하나도 안 먹었어 🥶"
* **Q1. 틀린 것 찾기:** 지방의 역할로 올바르지 않은 것은? (정답: 산소를 혈액으로 운반한다)
* **Q2. O/X 판별:** 좋은 지방과 나쁜 지방 구별하기. (정답: 아보카도, 고등어, 올리브오일은 좋은 지방!)
* **Q3. 금고 암호 해독 (방탈출):** 뒤섞인 6개의 글자 `산지화포불방`을 재조합하여 올바른 단어 찾기.
  * (정답: 불포화지방산)

### 🍊 방 4: 푸석 (비타민) → 코드 획득: [도]
* **대사:** "피부에 이상한 반점이 생기고 잇몸에서 피가 나... 과일이라곤 평생 먹어본 적 없어 😟"
* **Q1. 객관식:** 피부 반점 및 잇몸 출혈과 관련된 비타민은? (정답: 비타민 C 부족 - 괴혈병)
* **Q2. 탐정 매칭 गेम:** 비타민 A/C/D/B1과 그에 해당하는 결핍증(야맹증, 괴혈병, 구루병, 각기병) 올바르게 선 긋기.
* **Q3. 비타민 분류 미션 (방탈출):** 쏟아진 비타민 카드들을 클릭하여 분류하기.
  * (정답: 지용성은 A,D,E,K / 수용성은 B,C)

### 🥛 방 5: 창백 (무기질) → 코드 획득: [사]
* **대사:** "자꾸 넘어지고 이가 자꾸 빠져... 우유는 싫어서 평생 안 마셨어 🦷"
* **Q1. 객관식:** 뼈와 이를 튼튼하게 만드는 무기질은? (정답: 칼슘)
* **Q2. 장바구니 미션:** 창백의 뼈를 튼튼하게 할 식품 3개 담기 (정답: 우유, 멸치, 브로콜리)
* **Q3. 표 오류 찾기 (방탈출):** 무기질 정리표에서 잘못 적힌 한 줄 찾기.
  * (정답: 나트륨(Na)은 칼슘 흡수 촉진을 돕는다 → ❌ 나트륨은 세포 삼투압을 조절하며, 칼슘 흡수는 비타민 D/C/젖당이 돕습니다.)

### 💧 방 6: 갈증 (물) → 코드 획득: [!]
* **대사:** "입이 바짝 말라서 말도 못하겠어... 목이 마를 때만 물 마시면 된다고 생각했는데 💧"
* **Q1. 틀린 것 찾기:** 물의 역할로 틀린 것은? (정답: 에너지를 직접 만들어 낸다)
* **Q2. 탈수 진단:** 갈증의 증상을 보고 탈수 여부 판단하기 (정답: 소변 색 진해짐, 두통, 피부 탄력 감소 등은 탈수)
* **Q3. 자물쇠 풀기 (방탈출):** 수분 손실 퍼센트를 빈칸에 넣어 자물쇠 비밀번호 풀기.
  * (정답: 갈증 `2`%, 근육 피로 `4`%, 생명 위험 `2`0% → 비밀번호: **2420**)
